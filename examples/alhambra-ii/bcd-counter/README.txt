This example demonstrates a small project testbenches to test individual 
modules as well as the top module.

Some command to try:

apio lint
apio test
apio sim main_tb.v
apio report
apio build
apio upload            // Requires an alhambra-ii board
apio graph             // It generates _build/hardware.svg
apio graph reset_gen 
apio clean
apio format
